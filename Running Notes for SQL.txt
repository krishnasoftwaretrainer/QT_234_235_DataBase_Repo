1.DDL:Data Defination Language

Create:Create new database Table structure.
Syntax:create table tablename(Column1 datatype(size),Column2 datatype size(2),etc....);
Size:3:You can enter values upto 3digits.
1:0 to 9
2:0 to 99
3:0 to 999

Alter:
->Add New Column
Syntax:alter table tablename add columnname datatype(size);

->Modify existing column[datatype or size]
Syntax:alter table tablename modify columnname new datatype(size);

->Rename Column name 
Syntax:alter table tablename rename column oldcolumnname to newcolumnname;

alter rename:sub to sub2 

Create New table by using existing data table structure.
Syntax:create table newtablename as select * from oldtablename;

->Drop the Column
Syntax:alter table tablename drop column columnname;

Rename of Table:Rename the Table name.
Syntax:rename oldtablename to newtablename;

Drop:Drop the table.
It perminetly delete the table structure.
Syntax:drop table tablename;

2.DML:Data Manipulation Language

Insert:Insert/Store data/information in already created database table.

Type1:Insert all column values in sequence or order.
Syntax:insert into tablename values(value1,'value2',etc...);
EX:SQL> insert into stu1 values(101,'Krishna',99,98,97);
O/P: 1 row created.

Type2:Insert values into specfic columns.
Syntax: insert into tablename(Column1,Column2) values(value1,value2);

Type3:Insert all column values continues.
Syntax:insert into tablename values(&columnname1,&columnname2,...);
Column1 Value:
Column2 Value:
/
Type4:Insert same values continues.
Syntax:insert into tablename values(&&columnname1,&&columnname2,...);

Select:To select or view the inserted all records or specfic records or columns.
Syntax:select * from tablename;

Syntax2:update tablename set columnname='updatevalue' where condition;
